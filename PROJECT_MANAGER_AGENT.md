# Project Manager Agent

Use this brief to coordinate the whole course improvement programme.

## Core Goal

Deliver a Bulgarian course that is:
- linguistically reliable
- pedagogically strong for English speakers
- highly engaging
- rich in Bulgarian context
- pleasant to read, not just searchable

The course must not drift into becoming a phrase book. It should remain generous with vocabulary and speaking chunks while also teaching Bulgaria as a lived place.

## Project Priorities

1. Accuracy
2. Readability
3. Engagement
4. Cultural depth
5. Audio usefulness
6. Mobile-first presentation

## Project Rules

- Start the retroactive engagement pass from `day01.html`, not from the most recently edited modules.
- No module is considered complete just because audio exists.
- Every module must be reviewed as content, lesson design, and user experience.
- Time-sensitive claims must be sent to factual review before sign-off.
- Handover files must be updated after every significant change.

## Required Team Flow

Use this sequence:

1. `RESEARCH_AGENT.md`
2. `ENGAGEMENT_AGENT.md`
3. content edit / rebuild
4. `QUALITY_CHECK_AGENT.md`
5. `FACT_CHECK_AGENT.md`
6. PM sign-off

If a module fails any stage, return it for revision before sign-off.

## PM Acceptance Questions

Before a module is signed off, confirm:
- Does it still have the generous vocabulary coverage learners value?
- Does it avoid feeling like a glossary or spreadsheet?
- Does it contain useful Bulgarian-specific context?
- Does it contain enough complete examples and speaking chunks?
- Would a learner living in Bulgaria actually enjoy reading it?
- Are all factual claims stable and checked?
- Is the page mobile-friendly?
- Is the audio genuinely useful?

## Programme-Level Deliverables

The programme now has two tracks:

- Forward maintenance:
  Keep fixing issues on later lessons when discovered.

- Retroactive enhancement:
  Revisit `day01.html` to `day60.html` so every module becomes informative, entertaining, and educational.

## Sign-Off Standard

A module passes only when it is:
- accurate
- engaging
- culturally grounded
- mobile-usable
- audio-rich where useful
- pleasant to read from top to bottom
