# Course Review Roadmap

This file defines the next project phase.

## Phase Goal

Rebuild the full course so every lesson is:
- accurate
- engaging
- culturally rich
- generous with vocabulary and examples
- nice to read
- useful for people living in Bulgaria or planning to move there

## Strategic Clarification

The course should keep the wide phrase coverage that users love.

The problem to solve is not “too much content”.
The problem is “too much content without enough framing, context, and delight”.

So the task is:
- keep the useful vocabulary density
- add stronger structure
- add Bulgarian context
- add memorable facts and stories
- add higher-quality examples
- make every page feel like a lesson, not a reference sheet

## Team Structure

- `PROJECT_MANAGER_AGENT.md`
  Owns sequencing, standards, and sign-off.
- `RESEARCH_AGENT.md`
  Finds high-value Bulgarian content to enrich lessons.
- `ENGAGEMENT_AGENT.md`
  Ensures the lesson is interesting, human, and culturally alive.
- `QUALITY_CHECK_AGENT.md`
  Reviews language, pedagogy, CEFR fit, transliteration, and audio logic.
- `FACT_CHECK_AGENT.md`
  Verifies factual claims and removes risky or stale statements.

## Review Sequence Per Module

1. Research pass
2. Engagement design pass
3. Content rebuild
4. Audio review / audio additions
5. Language QA sweep
6. Factual QA sweep
7. PM sign-off

## Retroactive Review Order

Start from the beginning:

1. `day01.html` to `day05.html`
2. `day06.html` to `day10.html`
3. `day11.html` to `day16.html`
4. `day17.html` to `day20.html`
5. continue upward in sequence

Reason:
- the early lessons define first impressions
- they need the stronger engagement standard too
- the whole course should feel coherent from Day 1 onward

## What Every Module Should Gain

Each reviewed module should aim to include:
- a strong scene-setting opening before Section 1
- richer Bulgarian context
- more complete natural examples
- interesting facts or stories where relevant
- “how Bulgaria really works” detail
- mobile-friendly reading experience
- audio that supports the most important language

## Mandatory Sweep Types

Every module now needs:
- engagement sweep
- quality / language sweep
- fact-check sweep

No page should skip factual review if it contains real-world claims.

## Sign-Off Standard

A module is ready only when it is:
- informative
- entertaining
- educational
- credible
- easy to read
- useful in real Bulgarian life
